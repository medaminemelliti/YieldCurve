# Data

This folder is intentionally empty. Download the following files and place them here before running the notebook.

## 1. Daily yield curve snapshot (`data.xlsx`)
- Go to [tunisiayieldcurve.tn](https://www.tunisiayieldcurve.tn/public/)
- Select your valuation date and download the Excel file
- Rename it `data.xlsx`

## 2. BTCT reference file (`btct_ref.xlsx`)
- Go to [tunisieclearing.com](https://www.tunisieclearing.com/tc/fr/statistiques/encours)
- Download the BTCT outstanding instruments file
- Rename it `btct_ref.xlsx`

## 3. BTA reference file (`bta_ref.xlsx`)
- Same page as above
- Download the BTA outstanding instruments file
- Rename it `bta_ref.xlsx`
